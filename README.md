## 🏔️ 关于齿鸣幽境 — 新手必读

> 齿鸣幽境是一个以**结构探险 + 机械动力 + 养老种田**为核心玩法的 Minecraft 整合包服务器。  
> 你可以在这里深入各类神秘地下结构，搭建复杂的机械工厂，也可以安安静静地耕田养殖。  
> 无论你是哪种玩法偏好，都欢迎来这片幽境安家。

---

### 1. 下载整合包

整合包通过 **NeoForge** 加载，请按以下步骤获取：

1. 前往本仓库的 [**Releases 页面**](../../releases/latest)
2. 下载最新版本的整合包文件（`.zip` 格式）
3. 请确认你下载的版本号与当前服务端版本一致（见[更新记录](#-更新记录)）

> ⚠️ **注意**：请不要使用旧版整合包加入服务器，版本不匹配会导致无法连接。

---

### 2. 安装整合包

**前置要求：**
- Java **21** 或更高版本（[下载 Java](https://adoptium.net/)）
- 正版 Minecraft Java Edition（或使用支持离线登录的启动器，如 HMCL / PCL2）
- 内存建议：**至少 6 GB** 分配给游戏，8 GB 更佳

**安装步骤（以 HMCL / PCL2 为例）：**

1. 打开你的启动器，选择 **「导入整合包」**
2. 选择刚才下载的 `.zip` 文件
3. 等待启动器自动安装 NeoForge 及全部模组
4. 安装完成后，**不要修改任何模组文件**（如需添加自用模组请先联系管理员确认兼容性）

> 💡 **推荐启动器**：[PCL2](https://github.com/Hex-Dragon/PCL2) | [HMCL](https://hmcl.huangyuhui.net/)

---

### 3. 基本设置与加入服务器

**推荐游戏设置（降低卡顿）：**

| 设置项 | 推荐值 |
|--------|--------|
| 视距 | 8 ~ 10 |
| 模拟距离 | 6 ~ 8 |
| 图形 | 快速 |
| 流畅光效 | 开启 |
| 粒子效果 | 最少 |

**加入服务器：**

1. 启动游戏，进入 **多人游戏**
2. 点击 **添加服务器**
3. 服务器地址填写：

```
your.server.address.here
```

> 请将上方地址替换为实际服务器 IP。有疑问请联系管理员。

---

### 📜 服务器须知

1. **禁止任何形式的作弊、外挂、X-Ray**，违者永封。
2. **禁止恶意破坏他人建筑**，请尊重每一位玩家的劳动成果。
3. **禁止刷屏、辱骂、人身攻击**，保持友好的交流环境。
4. **大型机械装置请控制规模**，避免过度占用服务器性能（TPS 友好服务器，请不要刷怪塔超过 3 层）。
5. **未经管理员许可，禁止私自修改服务端相关配置**。
6. 遇到 Bug 请在 [Issues](../../issues) 中汇报，而不是自行利用。
7. 服务器最终解释权归管理团队所有。

## ⚙️ 模组介绍

> 以下是整合包核心模组的简要介绍，帮助你快速了解服务器的主要玩法内容。  
> 完整模组列表见 `/config` 目录下的 `modlist.txt`。

---

### 探索 & 结构类

| 模组名称 | 简介 |
|----------|------|
| **Alex's Caves** | 新增大量独特地下洞穴生态群系，每个洞穴都有专属生物与资源 |
| **Dungeons and Taverns** | 在世界各地生成大量随机结构，包括地牢、酒馆、哨塔等 |
| **YUNG's Better Dungeons** | 大幅强化原版地牢的质量与规模，值得一探 |
| **Structory** | 添加各类小型但精心设计的结构，遍布主世界各地 |
| **When Dungeons Arise** | 超大型空中/地下建筑结构，高难度高收益 |

> 💡 *占位：可按实际整合包内容替换以上模组列表*

---

### 机械 & 动力类

| 模组名称 | 简介 |
|----------|------|
| **Create** | 整合包核心之一，以旋转与运动为基础的机械动力模组 |
| **Create: Steam 'n' Rails** | Create 的铁路扩展，建造你的蒸汽铁路网络 |
| **Create: Slice & Dice** | Create 与农业模组的联动扩展 |
| **Pneumaticraft** | 气压驱动的机械与自动化系统 |
| **Mekanism** | 高科技矿物处理与能源系统 |

> 💡 *占位：可按实际整合包内容替换以上模组列表*

---

### 农业 & 种田类

| 模组名称 | 简介 |
|----------|------|
| **Farmer's Delight** | 扩展大量新食物、烹饪与农业内容 |
| **Pam's HarvestCraft 2** | 数百种新作物与食谱，养老种田必备 |
| **Beekeeping** | 深度养蜂系统，生产蜂蜜与蜂蜡相关物品 |
| **Aquaculture 2** | 扩展钓鱼系统，新增大量鱼类与钓鱼相关内容 |

### 优化 & 辅助类

| 模组名称 | 简介 |
|----------|------|
| **Embeddium** | 显著提升渲染性能（Sodium 的 NeoForge 移植版） |
| **FerriteCore** | 大幅降低内存占用 |
| **Starlight** | 重写光照引擎，减少卡顿 |
